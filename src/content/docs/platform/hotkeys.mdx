Use keyboard shortcuts to move around the dashboard and open common actions without reaching for the mouse.

Most navigation shortcuts are **key sequences**: press the first key, then the second (for example, `G` then `S`). They only work when your focus is not inside a text field.

<Aside variant="tip">
	Hotkeys are still being expanded. Some shortcuts may not work on every page
	yet. If you have suggestions, share them on our Discord server.
</Aside>

## Project navigation

Available while viewing a project. Press `G`, then the second key.

| Hotkey  | Description                                 |
| ------- | ------------------------------------------- |
| `G` `M` | Open the project overview (main dashboard). |
| `G` `S` | Open project settings.                      |
| `G` `R` | Open retention.                             |
| `G` `F` | Open funnels.                               |
| `G` `V` | Open web vitals.                            |
| `G` `E` | Open error tracking.                        |
| `G` `U` | Open users.                                 |
| `G` `D` | Open downloads.                             |
| `P`     | Open the project switcher.                  |

## Global shortcuts

| Hotkey    | Description         |
| --------- | ------------------- |
| `Mod` `B` | Toggle the sidebar. |

`Mod` is `⌘` on macOS and `Ctrl` on Windows and Linux.

## Dashboard editing

On a custom project dashboard (not the default dashboard):

| Hotkey | Description                                 |
| ------ | ------------------------------------------- |
| `E`    | Toggle layout edit mode.                    |
| `C`    | Create a new chart (when not in edit mode). |
