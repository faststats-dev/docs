## SDKs

| SDK                       | Description                                                          |
| ------------------------- | -------------------------------------------------------------------- |
| [Java SDK](/java)         | Java & Kotlin projects: Bukkit, Paper, Velocity, Minestom, and more. |
| [Web SDK](/web-analytics) | Add analytics to your website with a lightweight script.             |

## Features

| Feature                                    | Description                                                            |
| ------------------------------------------ | ---------------------------------------------------------------------- |
| [Error Tracking](/platform/error-tracking) | Capture runtime errors with stack traces and sourcemap support.        |
| [Feature Flags](/platform/feature-flags)   | Roll out boolean and variation flags with targeting and opt-in access. |
| [Web Vitals](/platform/web-vitals)         | Monitor LCP, CLS, INP, FCP, and TTFB in the field.                     |
| [Chart Editor](/platform/chart-editor)     | Build chart queries visually with flow nodes and multi-series outputs. |
| [Retention](/platform/retention)           | Analyze cohort behavior and track returning users over time.           |
