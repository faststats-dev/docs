FastStats allows you to track errors in your application.  
For this, you have to enable <b>Error Tracking</b> in your project settings under <b>Features</b>.

## Data Retention

Error tracking data is retained for <b>30 days</b>.  
After this period, all stored error events are automatically deleted and can no
longer be accessed.
