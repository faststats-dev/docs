The Flow Editor lets you build chart queries by connecting nodes in a pipeline. Drag nodes onto the canvas, configure them, and wire them together. Data flows from top to bottom, starting at a **Data Source**, passing through transformations, and ending at an **Output**.

## Getting started

1. Add a **Data Source** node to choose which adapter and field to read from
2. Connect transformation nodes like **Filter**, **Group By**, or **Aggregate** to shape the data
3. Finish with an **Output** node to produce the chart result

Each node has handles on its top and/or bottom edge. Connect the bottom handle of one node to the top handle of the next to build the pipeline. The colored badge on each node shows its category. Sources, filters, aggregates, groupings, transforms, time operations, and outputs each have their own color.

### Group By

<div className="flex items-start gap-8 py-4 not-prose">
	<div className="flex-1 min-w-0 space-y-3">
		<p>
			Groups rows by one or more categorical fields. Usually paired with an{" "}
			<strong>Aggregate</strong> node to compute per-group metrics (for example,
			page views grouped by browser).
		</p>
		<ul className="list-disc pl-5 space-y-1 text-sm">
			<li>
				<strong>Fields</strong>: the grouping dimensions. Rows that share the
				same values across these fields end up in the same group. Add as many as
				you need.
			</li>
			<li>
				<strong>Multi-field mode</strong>: only shown when more than one field
				is selected.
				<ul className="list-disc pl-5 mt-1 space-y-0.5">
					<li>
						<code>combined</code>: joins field values into a single composite
						key (for example, <code>Chrome / Germany</code> as one group).
					</li>
					<li>
						<code>flat</code>: keeps each field as its own dimension so the
						chart can break down by either axis.
					</li>
				</ul>
			</li>
		</ul>
	</div>

### Time Group

<div className="flex items-start gap-8 py-4 not-prose">
	<div className="flex-1 min-w-0 space-y-3">
		<p>
			Groups rows into time-based buckets. This is the foundation of every
			time-series chart.
		</p>
		<ul className="list-disc pl-5 space-y-1 text-sm">
			<li>
				<strong>Bucket size</strong>: the interval each row rolls into:{" "}
				<code>Auto</code>, <code>Minute</code>, <code>Hour</code>,{" "}
				<code>Day</code>, <code>Week</code>, <code>Month</code>.{" "}
				<code>Auto</code> picks a sensible size based on the chart's time range.
			</li>
			<li>
				<strong>Missing buckets</strong>: what to do when a time period has no
				data:
				<ul className="list-disc pl-5 mt-1 space-y-0.5">
					<li>
						<code>Zero</code>: insert a zero value so the line stays continuous.
					</li>
					<li>
						<code>None</code>: leave the gap empty and the series breaks.
					</li>
				</ul>
			</li>
			<li>
				<strong>Cumulative</strong>: carry each bucket's value into every later
				bucket. Use this for running totals, such as total users or downloads so
				far.
			</li>
		</ul>
	</div>

### Text Transform

<div className="flex items-start gap-8 py-4 not-prose">
	<div className="flex-1 min-w-0 space-y-3">
		<p>
			Rewrites string values before they are grouped, filtered, or displayed.
			Useful for normalizing inconsistent text (for example, casing) or
			extracting derived labels.
		</p>
		<ul className="list-disc pl-5 space-y-1 text-sm">
			<li>
				<strong>Operation</strong>: which transformation to apply:
				<ul className="list-disc pl-5 mt-1 space-y-0.5">
					<li>
						<code>lowercase</code> / <code>uppercase</code> /{" "}
						<code>title case</code>: change casing
					</li>
					<li>
						<code>trim</code>: strip leading and trailing whitespace
					</li>
					<li>
						<code>append</code>: add a fixed string to the end of each value
					</li>
					<li>
						<code>replace text</code>: find an exact string and replace it
					</li>
					<li>
						<code>replace regex</code>: match a regular expression and
						substitute (use <code>\1</code>, <code>\2</code>, and so on for
						capture groups)
					</li>
					<li>
						<code>length</code>: replace each value with its character count
					</li>
				</ul>
			</li>
			<li>
				<strong>Text to append</strong>: only for <code>append</code>. The
				string added to every value.
			</li>
			<li>
				<strong>Find / Pattern</strong> and <strong>Replace with</strong>: only
				for <code>replace text</code> and <code>replace regex</code>. Leave the
				replacement empty to delete matches.
			</li>
		</ul>
	</div>
	<div>
		<TextTransformNodePreview />
	</div>
