Use the FastStats REST API to manage projects, dashboards, charts, and analytics data programmatically.
Create an API key in your user or organization settings.

<Callout type="warning">
	The API is currently in version 0. We may make breaking changes to the API in
	the future.
</Callout>

<Callout type="warning">
	Rate limits are still being adjusted. We may change them in the future.
</Callout>

The OpenAPI spec is available at [api.faststats.dev/openapi.json](https://api.faststats.dev/openapi.json).
